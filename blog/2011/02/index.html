目录结构

名称 解释
/ 根目录
bin 放普通用户的命令
boot linux的核心文件 如mrb镜像 grub启动
dev 硬件 如串口
etc 系统配置文件 如config
lib 存放动态库
mnt 挂载不同设备
proc 存放cpu 内存 硬件信息
sbin 存放超级用户命令
tmp 临时名录 放置临时信息
usr 类似windows的program file 内有src(source),bin,sbin
var 存放变动性信息 log spool cache
